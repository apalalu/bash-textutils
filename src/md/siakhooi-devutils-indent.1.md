# NAME
indent - indent text with a prefix.

# SYNOPSIS
**indent** prefix

# DESCRIPTION
Programs that indent each lines of text with a prefix.

# EXAMPLES
$ cat filename1 | **indent** '>>>>\ '  
\>\>\>\> This is line 1 in filename1  
\>\>\>\> This is line 2 in filename1
