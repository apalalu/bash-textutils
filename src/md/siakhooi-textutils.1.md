# NAME
**prefix** - add a prefix to all lines of text.\
**indent** - add a prefix to all lines of text.\
**suffix** - add a suffix to all lines of text.


# SYNOPSIS
**prefix** prefix\
**indent** prefix\
**suffix** suffix

# DESCRIPTION
Collection of programs to tranform line of texts.
